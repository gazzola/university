Máquina de Turing na unha

Implemente um programa que execute uma Máquina de Turing arbitrária. Para isso, o programa deverá receber como entrada a especificação de uma MT e a string de entrada e devolver como saída a fita resultante e se a string é aceita ou não (isso implica que a MT deve decidir a linguagem; não se preocupe em testar MTs que apenas aceitam uma linguagem).

A especificação da MT pode ser lida de um arquivo, e deverá conter o necessário: conjunto de estados, alfabeto de entrada, lista de transições, etc.

Considere também que a MT possui uma fita semi-infinita: infinita para a direita. Sinta-se livre para implementar qualquer das extensões mais comuns de MTs.

This is a work for the formal languages class on graduation over computer science.
